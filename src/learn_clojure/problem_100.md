# Problem 100, Least Common Multiple

**Difficulty:** easy

Write a function which calculates the least common multiple. Your function should accept a variable number of positive integers or ratios.

```clj
(== (__ 2 3) 6)
```

```clj
(== (__ 5 3 7) 105)
```

```clj
(== (__ 1/3 2/5) 2)
```

```clj
(== (__ 3/4 1/6) 3/2)
```

```clj
(== (__ 7 5/7 2 3/5) 210)
```

# Problem 32, Duplicate a Sequence

**Difficulty:** easy

Write a function which duplicates each element of a sequence.

```clj
(= (__ [1 2 3]) '(1 1 2 2 3 3))
```

```clj
(= (__ [:a :a :b :b]) '(:a :a :a :a :b :b :b :b))
```

```clj
(= (__ [[1 2] [3 4]]) '([1 2] [1 2] [3 4] [3 4]))
```

```clj
(= (__ [44 33]) [44 44 33 33])
```

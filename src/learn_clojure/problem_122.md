# Problem 122, Read a binary number

**Difficulty:** easy

Convert a binary number, provided in the form of a string, to its numerical value.

```clj
(= 0     (__ "0"))
```

```clj
(= 7     (__ "111"))
```

```clj
(= 8     (__ "1000"))
```

```clj
(= 9     (__ "1001"))
```

```clj
(= 255   (__ "11111111"))
```

```clj
(= 1365  (__ "10101010101"))
```

```clj
(= 65535 (__ "1111111111111111"))
```
